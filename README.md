# Course Compass

A full-stack web application that aggregates online programming courses and allows users to search and filter them.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Search courses
- Filter by language, price, rating
- Real-time results
- Responsive UI

## How to Run
Frontend:
npm run dev

Backend:
npm run dev
